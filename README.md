neovim-cutlass
==============

This Neovim plugin redirects the delete operations to the black hole register
and provides a convenient way to create a new mapping for the cut operation.

Related plugins
===============

* [vim9-cutlass](https://github.com/Coacher/vim9-cutlass) for Vim 9.
  Actively maintained.

* [vim-cutlass](https://github.com/svermeulen/vim-cutlass) for Vim 8.
  Unmaintained.

* [vim-easyclip](https://github.com/svermeulen/vim-easyclip) for Vim 7.
  Unmaintained.
